# PARSER - ICS-Guard

Chứa Log Parser phục vụ chuẩn hóa các loại log đầu vào thô (Syslog, JSON, CSV) thành định dạng chuẩn chung (Unified Format).

## 👤 Thành viên phụ trách
*   **Tòng Quốc Hưng**

## 🚀 Nhiệm vụ & Mục tiêu triển khai
- Giai đoạn 1: Phân tích cấu trúc logs thô để lên thiết kế chuẩn hóa định dạng.
- Giai đoạn 2: Code Log Parser để chuẩn hóa dữ liệu log đầu vào trước khi đưa vào Analysis.
